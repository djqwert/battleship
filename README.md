# battleship
A simple battleship game developed for Computer network course.

# Installation
To build project:

```sh
make
```
To execute server:

```sh
./battle_server <socket_id_server>
```

To execute clients:

```sh
./battle_client <ip_server> <socket_id_server>
```

If you are running server and clients in local choose ip: 127.0.0.1
